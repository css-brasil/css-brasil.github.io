Seguuura coração, o CSS Brasil chegou!

Escrito em September 1, 2016

Com exemplos fantásticos e artigos incríveis, criamos o CSS Brasil com a expectativa de compartilhar conhecimento e promover talentos.

Faaaaaala galeeeera hoje é um dia histórico, pois este é o nosso primeiro post de muiiiitos do CSS Brasil \o/

Algumas semanas atrás, entrei em algumas discussões bastante empolgantes sobre css e um dos "temas abordados" foi o que o CSS é capaz de fazer. Logo entrei no meu codepen e fui demonstrando algumas técnicas css que utilizo bastante, como a replicação de formas com box-shadow e a mágica do :checked. Foi quando me perguntaram se existia algum site/blog no brasil no estilo Codrops. Indiquei alguns blogs brasileiros que gosto bastante, mas o único que se tratava apenas de CSS, era o do maujor.

Alguns dias depois, navegando por alguns blogs de evangelistas CSS internacionais, notei que não existe nenhum blog brasileiro exclusivo só para CSS, como as outras linguagens tem (javascript por exemplo tem o jsrocks.org). Então tive a ideia show, muito foda, da hora ideia de criar uma espécie de Codrops brasileiro, com vários artigos e exemplos fantásticos!

Depois do CSS Brasil, a forma que o CSS é visto por muitos dev's, nunca mais será a mesma :D

Um pouquinho sobre o blog

Iremos dividir o nosso blog em 2 partes (por enquanto...)

  • Artigos
  • Playground

    Em artigos será onde iremos postar vários assuntos técnicos, sempre relacionados a CSS. Dentro de cada post teremos um botão para visualizar seu exemplo e/ou um botão para baixar o código fonte.

    Em playground teremos vários exemplos incríveis de frameworks ou algum projeto feito só em CSS.

    O melhor é que nosso blog é todo open source e está disponível no github, caso você queira entrar para a família do CSS Brasil.

Não vou me prologar muito neste post, pois ainda teremos um próximo, explicando a formatação e etc...

Então, é isto galera, esse post foi só para agitar um pouquinho nosso final de semana e deixar vocês ligados nos próximos dias para o nosso primeiro artigo técnico \o/\o/

Um forte abraço!
